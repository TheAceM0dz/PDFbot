Português:
PDFBOT é um conversor de arquivos para PDF (e vice-versa) feito em Python, criado para rodar 100% no Termux, direto do celular Android — sem precisar de computador. Além da conversão básica (DOCX, TXT, Markdown, HTML e imagens para PDF, e PDF para PNG, JPEG, DOCX e TXT), o bot inclui ferramentas avançadas como unir PDFs, comprimir, extrair páginas específicas, proteger com senha, adicionar marca d'água e OCR para tornar PDFs escaneados pesquisáveis. Tem conversão em lote, histórico de uso, interface interativa colorida (Rich + Questionary) e suporte a português e inglês.


English:
PDFBOT is a Python-based file-to-PDF converter (and vice versa), built to run entirely on Termux, straight from an Android phone — no computer required. Beyond basic conversion (DOCX, TXT, Markdown, HTML, and images to PDF, plus PDF to PNG, JPEG, DOCX, and TXT), it includes advanced tools like merging PDFs, compression, extracting specific pages, password protection, watermarking, and OCR to make scanned PDFs searchable. It also supports batch conversion, usage history, a colorful interactive interface (Rich + Questionary), and both Portuguese and English languages
